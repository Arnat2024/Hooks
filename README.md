# React + Vite

Цей репозиторій містить приклади використання всіх хуків, згаданих на лекції. Всі відповідні компоненти імпортовані, але частина закоментована.

Нагадую, що для запуску прєкту потрібно завантажити файли з репо і виконати наступні команди
npm install - для встановлення залежностей проєкту
npm run dev - для запуску проєкту
